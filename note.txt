DOM(Documnet object model)

THE browser creates a dom model(data tree) of the program that is being run.Javascript is made dynamic using these structures.

dom methods
-----------
1-document.getElementsByTagName('h1')

2-document.getElementById("id")

3-document.getElementsByClassName('class name')

4-document.querySelector('tagname/#id/.class')-single element only

5-document.querySelectorAll('tagname/#id/.class')-all elements
